# compare_execution


月別内訳を常に表示する専用スクリプトを使う場合:

```bash
python3 scripts/gha_compare_execution_monthly.py \
  --owner your-org \
  --repo your-repo \
  --start-1 2025-01-01 --end-1 2025-03-31 \
  --start-2 2026-01-01 --end-2 2026-03-31 \
  --max-duration-hours 24
```
